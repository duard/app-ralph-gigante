# TGFCAB - Cabeçalhos de Notas Fiscais

## Descrição
TGFCAB é a tabela principal do Sankhya ERP para cabeçalhos de notas fiscais. Representa o documento fiscal que evolui para NF-e (Nota Fiscal Eletrônica). Contém todas as informações essenciais sobre transações fiscais, incluindo valores, datas, partes envolvidas e configurações tributárias.
